# ChocoOps API Documentation

## Overview
This API follows RESTful principles.

## Endpoints
(Run `npm run dev` and visit `/api-docs` for the interactive Swagger UI)

### Auth
- `POST /api/auth/login`
- `POST /api/auth/register`

### Inventory
- `GET /api/inventory`
- `POST /api/inventory`

### Manufacturing
- `POST /api/production/batches`
